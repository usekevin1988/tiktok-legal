Privacy Policy

Last updated: February 2026

1. Overview

Nexus ("the Application") is a personal, locally-hosted video automation tool. It does not collect, store, or process personal data from any third party.

2. Data Collection

The Application does not collect any personal data from external users because it has none. The only user is the operator themselves.

3. Data Stored Locally

The following data is stored exclusively on the operator's local machine:

  • API credentials (TikTok, YouTube, OpenAI, Gemini) for authenticated access
  • OAuth tokens for platform integration
  • Generated video files, scripts, and metadata
  • Upload history and scheduling data

No data is transmitted to any server other than the respective API endpoints required for video publishing.

4. Third-Party Services

The Application communicates with:

  • TikTok API – for video uploads (governed by TikTok's Privacy Policy)
  • YouTube Data API – for video uploads (governed by Google's Privacy Policy)
  • OpenAI API – for text-to-speech generation
  • Google Gemini API – for content generation

Each service's own privacy policy governs data sent to their servers.

5. No Tracking

The Application does not use cookies, analytics, tracking pixels, or any form of user monitoring.

6. Contact

For questions regarding this policy, contact the operator directly.
